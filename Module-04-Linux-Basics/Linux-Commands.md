# Linux Commands

## ls

Lists files and directories.

Example:
ls

## pwd

Shows current working directory.

Example:
pwd

## cd

Changes directory.

Example:
cd Documents

## mkdir

Creates a new directory.

Example:
mkdir projects

## cp

Copies files.

Example:
cp file1.txt file2.txt

## mv

Moves or renames files.

Example:
mv old.txt new.txt

## rm

Removes files.

Example:
rm file.txt

## cat

Displays file contents.

Example:
cat file.txt

## grep

Searches text inside files.

Example:
grep "error" app.log

## chmod

Changes file permissions.

Example:
chmod 755 script.sh

## ps

Displays running processes.

Example:
ps

## top

Shows live system resource usage.

Example:
top

## ssh

Connects to a remote server securely.

Example:
ssh user@server-ip
