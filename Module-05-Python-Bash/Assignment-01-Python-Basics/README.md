# Python Basics - Assignment 01

## Course
TuteDude DevOps Course

## Module
Python and Bash

## Objective

The objective of this assignment is to understand the basic concepts of Python programming, including conditional statements, dictionaries, and file handling.

## Programs Implemented

### 1. Grade Checker
- Accepts marks as input.
- Displays grades using if-elif-else.

### 2. Student Grades Dictionary
- Adds student names and grades.
- Updates existing student grades.
- Displays all student records.

### 3. Write to a File
- Creates a text file.
- Writes data into the file.

### 4. Read from a File
- Reads data from the created file.
- Displays the contents on the screen.
