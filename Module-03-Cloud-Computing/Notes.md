# Introduction to Cloud Computing

## What is Cloud Computing?

Cloud Computing is the delivery of computing services such as servers, storage, databases, networking, software, and analytics over the internet.

Instead of buying physical hardware, users can rent resources on demand.

## Why Cloud Computing?

* Pay only for what you use
* High scalability
* Global accessibility
* High availability
* Reduced infrastructure costs

## Cloud Service Models

### IaaS (Infrastructure as a Service)

Provides virtual machines, storage, and networking.

Examples:

* AWS EC2
* Azure Virtual Machines

### PaaS (Platform as a Service)

Provides platforms for application development.

Examples:

* AWS Elastic Beanstalk
* Google App Engine

### SaaS (Software as a Service)

Provides software over the internet.

Examples:

* Gmail
* Google Drive
* Microsoft 365

## Deployment Models

### Public Cloud

Shared cloud infrastructure.

Example:

* AWS
* Azure
* Google Cloud

### Private Cloud

Cloud resources used by a single organization.

### Hybrid Cloud

Combination of public and private cloud.

## Benefits of Cloud

* Scalability
* Flexibility
* Cost savings
* Reliability
* Disaster recovery
