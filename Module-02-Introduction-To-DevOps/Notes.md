# Introduction to DevOps

## What is DevOps?

DevOps is a culture and set of practices that combines Development and Operations teams to improve software delivery speed, quality, and reliability.

## Goals of DevOps

* Faster software delivery
* Better collaboration
* Automation
* Continuous feedback
* Reduced deployment failures

## DevOps Lifecycle

1. Plan
2. Code
3. Build
4. Test
5. Release
6. Deploy
7. Operate
8. Monitor

## Benefits of DevOps

* Faster releases
* Improved quality
* Better communication
* Reduced manual work
* Higher reliability

## Common DevOps Tools

| Stage            | Tool       |
| ---------------- | ---------- |
| Code             | Git        |
| Build            | Maven      |
| CI/CD            | Jenkins    |
| Containerization | Docker     |
| Orchestration    | Kubernetes |
| Cloud            | AWS        |
| Monitoring       | Prometheus |
